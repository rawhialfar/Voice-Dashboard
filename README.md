# Voice-Dashboard
Full-stack voice agent dashboard with React/TypeScript frontend and Express backend.
```
Frontend: React + Vite + TypeScript
Backend: Express + TypeScript
CMS: Payload CMS
```

## Quick Start

### 1. Clone the Repository
```powershell
git clone https://github.com/rawhialfar/Voice-Dashboard.git
cd Voice-Dashboard
```

### 2. Install Dependencies & Run the Project

Open **two PowerShell terminals** in **VS Code** (or two terminals on your system):

**Terminal 1 - Backend:**
```bash
cd backend
npm install
npm run dev
```

Runs the Express backend on **[http://localhost:8000](http://localhost:8000)**


**Terminal 2 - Frontend:**
```bash
cd frontend
npm install
npm run dev
```

Runs the Vite React frontend on **[http://localhost:5173](http://localhost:5173)**
